# Ping Monitor PRO

🌍 Öffentlicher Ping Monitor mit Statusüberwachung und Speicherung

## Funktionen
- Websites pingbar (echte HTTP/HTTPS Requests)
- Statusanzeige: online/offline
- Intervall einstellbar (ms)
- Start/Stop / Löschen von URLs
- Live-Update für alle Besucher

## Installation

```bash
git clone <dein-repo>
cd ping-monitor-pro
npm install
npm start
